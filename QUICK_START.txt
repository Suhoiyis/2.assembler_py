Q: 编译失败？
A: 检查 g++ --version（需要 5.0+）
   Ubuntu: sudo apt-get install build-essential
   macOS: brew install gcc

Q: 测试不通过？
A: 运行 python3 run_tests.py 查看详细错误
   检查 build/iss 是否存在

Q: 无限循环怎么办？
A: 按 Ctrl+C 中断
   检查汇编代码的停机指令

Q: 如何集成到 Windows.py？
A: 查看 ISS_WINDOWS_INTEGRATION.md
   快速版本 2 小时，完整版本 4-6 小时
